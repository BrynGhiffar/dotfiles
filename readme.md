## I use ...
* [Kitty](https://github.com/kovidgoyal/kitty) as my terminal emulator
* [Nvim](https://github.com/neovim/neovim) as my text editor
* [Hyprland](https://github.com/hyprwm/Hyprland) as my window manager
* [Waybar](https://github.com/Alexays/Waybar)
* [Fzf](https://github.com/junegunn/fzf) as my [app switcher](/scripts/fzfapp)
* [Nushell](https://github.com/nushell/nushell) as shell (although I still use bash)
